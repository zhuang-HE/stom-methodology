# Workflow Templates — 详细任务模板

> 从 SKILL.md 主文件移出以节省 token。按需读取。

## 模板使用原则

不要直接从接口想起，而要从任务模板想起。每个模板定义了：适用场景 → 默认流程 → 输出标准。

---

### 1. 单标的行情分析

**适用**：
- 看下 XX 最近怎么样
- 这票最近强不强
- 今年以来表现如何

**默认流程**：
1. 解析标的（代码/名称 → ts_code）
2. 确定时间范围（"最近" → 近 20T，"今年以来" → YTD）
3. 取行情 + daily_basic（PE/PB/换手率）
4. 总结区间涨跌、成交活跃度、高低点、波动率
5. **输出**：一句话结论 + 关键数字表

---

### 2. 多标的横向对比

**适用**：
- XX 和 YY 谁更强
- 把这几家公司对比一下

**默认流程**：
1. 锁定对象（统一为 ts_code 列表）
2. 统一时间口径（所有标的相同区间）
3. 选 3～5 个关键指标（涨幅、波动、估值、成交量）
4. 输出对比表（按主指标排序）
5. **输出**："谁在哪方面更强"的总结

---

### 3. 财务质量快照

**适用**：
- 看下 XX 财报
- 最近几个季度利润趋势
- 财务质量怎么样

**默认流程**：
1. 拉最近 8 个季度 + 最近年度财务核心数据
2. 区分营收、利润、毛利率、ROE、现金流五个维度
3. 标出改善(↑)/恶化(↓)/波动(↔)点
4. 说明累计值 vs 单季值 vs 同比口径
5. **输出**：趋势判断 + 异常标注表

---

### 4. 估值分析 / 筛选

**适用**：
- 现在估值高不高
- 谁更便宜
- 筛低估值高股息

**默认流程**：
1. 明确标的池（单票 / 板块 / 自选列表）
2. 拉 daily_basic（PE/PB/股息率/市值）+ fina_indicator（ROE等）
3. 必要时联动财务质量做交叉验证
4. 输出排序、极值、口径说明
5. **输出**：估值分位表 + 偏高/偏低/合理标签

---

### 5. 资金流追踪

**适用**：
- 最近资金在买什么
- 北向最近流向哪里
- 主力资金流入最多的是谁

**默认流程**：
1. 明确资金口径（北向 / 主力 / 龙虎榜 / 板块资金）
2. 确定时间窗（近5T/近20T/近月）
3. 拉净流入、活跃成交、持续性指标
4. 和价格表现联动解释（资金驱动? 跟随? 背离?）
5. ⚠️ 避免把单日噪声说成趋势
6. **输出**：流向排行 + 关键异动解读

---

### 6. 板块 / 题材轮动分析

**适用**：
- 最近哪个板块最强
- 机器人最近强在哪
- 某概念板块里有哪些成分股

**默认流程**：
1. 确定分类口径（申万行业 / 同花顺概念 / 中信）
2. 拉板块区间表现（涨跌幅、量能、领涨标的）
3. 必要时联动成分股详情、资金流、涨停梯队
4. 输出强势板块排行与代表标的
5. **输出**：板块热度表 + 轮动方向判断

---

### 7. 公告 / 新闻 / 事件梳理

**适用**：
- 最近有什么公告
- 有没有什么催化
- 最近新闻面怎么样

**默认流程**：
1. 明确对象和时间窗（个股/板块 + 近7天/近30天）
2. 拉 anns_d / news / research_report / major_news
3. 去噪，提炼 3～5 条主线（政策催化 / 业绩驱动 / 事件驱动 / 主题炒作）
4. 区分事实（公告）、确认（研报）、推测（媒体解读）
5. 必要时结合股价异动做弱因果解释（不强行关联）
6. **输出**：时间线摘要 + 催化评级

---

### 8. 数据导出与研究准备

**适用**：
- 拉一份 CSV
- 做回测数据表
- 导出某段时间的行情/财务数据

**默认流程**：
1. 明确数据范围、频率、字段需求
2. 采用分段策略取数（长区间按年/季度切片）
3. 清洗、去重、统一字段类型
4. 输出 CSV / parquet
5. 附元信息：接口名、参数、拉取时间、行数、字段列表
6. **输出**：数据文件路径 + 质量报告

---

### 9. 综合研究简报（Investment Brief）

**适用**：
- 给我快速研究一下 XX
- 做个投资者视角简报
- 先给个全景判断

**默认流程**（这是最完整的模板）：

```
1. 一句话结论（看多/中性/看空 + 核心理由）
2. 行情表现（近期走势、相对强弱、量能变化）
3. 财务趋势（营收/利润/质量指标走向）
4. 估值水平（历史分位、同行对比）
5. 资金流情况（主力/北向/机构动向）
6. 公告 / 新闻催化（近期关键事件）
7. 风险点（下行风险列示）
8. 值得继续深挖的问题（下一步研究方向）
```

**输出格式**：结构化 Markdown 报告，适合直接阅读。
